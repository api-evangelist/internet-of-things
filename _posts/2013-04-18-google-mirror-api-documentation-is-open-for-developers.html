The Google Mirror API allows you to build web-based services, called Glassware, that interact with Google Glass.It provides this functionality over a cloud-based API and does not require running code on Glass.We announced the Mirror API here.The Google Mirror API documentation is now open on Google Developers.The quick start is open for two languages – Python and Java and code is also available on Github.
