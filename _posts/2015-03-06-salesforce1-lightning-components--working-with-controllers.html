Guest post: Peter Knolle is a Solutions Architect at Trifecta Technologies, Force.com MVP, and Salesforce Developer Group Organizer.  He holds six Salesforce certifications, blogs regularly at peterknolle.com, and is active on Twitter @PeterKnolle.Salesforce1 Lightning Components includes a framework (the Lightning Component Framework) that allows developers to define client-side controllers with methods that respond to events such as user interactions or lifecycle events and server-side controllers with actions that run on the server and perform processing such as database interactions.
